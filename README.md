# Stillframe-blog-site
A responsive blog website built with HTML, CSS, PHP, and MySQL. It supports user authentication, and allows users to create, edit, and manage blog posts through a clean, minimalist interface.


## Demo

https://stillframe.unilodge.live/

## Features

- User registration & login (with password hashing)
- Add, edit, publish blog posts (with cover images)
- Responsive design (Tailwind CSS)
- Modern and minimalist, user friendly UI
- Secure session management using PHP


## Stack

- PHP (backend logic)
- MySQL (database)
- Tailwind CSS (styling)
- XAMPP (local development)
- AWS (hosting)
- composer (dependencies)
